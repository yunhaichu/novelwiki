# Outline Generation Prompt

Use this prompt before chapter briefs and prose drafting.

The goal is not to write prose. The goal is to create a structured outline that can control later chapter briefs.

## Required Inputs

- genre or neutral project direction
- target length or first-volume length
- protagonist seed
- core constraint or premise
- forbidden patterns
- style goals
- comparable structure notes if available

## Output Object

Produce a structured outline with these sections:

```text
Project title:
Genre / story engine:
One-sentence core conflict:
Opening imbalance:
Protagonist external goal:
Protagonist inner pressure:
Primary obstacle:
Failure cost:
Three-layer suspense:
- immediate:
- midterm:
- longterm:
Five turning points:
- imbalance:
- first tool or method:
- relationship reconfiguration:
- truth turn:
- terminal choice:
Core relationship matrix:
Volume 1 goal:
Volume 1 end state:
Volume 1 required beats:
Consistency risks:
Forbidden drift:
```

## Rules

- Do not write chapter prose.
- Do not create decorative setting names unless needed.
- Do not overload the opening with world explanation.
- The first strong hook must appear in chapters 1-2.
- The first three chapters must contain at least three state changes.
- Every major setting rule must change a character choice.
- Every core relationship must include surface relation and hidden tension.
- Long-term mysteries must be distributed into immediate, midterm, and longterm layers.

## Evaluation Fields

Also include:

```text
Hook strength:
Goal clarity:
Relationship tension:
Cultivation / career / main-system relevance:
Expansion capacity:
Biggest risk:
```

Use short, concrete wording.
