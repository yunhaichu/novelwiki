# Chapter State Update Prompt

Status: DEPRECATED.

Do not use this prompt in the current workflow.

Use instead:

```text
prompts/05_wiki_sync_after_chapter.md
```

Reason:

`05_wiki_sync_after_chapter.md` is the current canonical post-chapter sync prompt. It includes the newer required chapter-state fields:

- world / civilization trend progress;
- Earth status progress when relevant;
- protagonist final-form progress;
- reader hook / payoff delivered;
- reader reward delivered;
- reader debt;
- pressure clock;
- repetition risk;
- next chapter constraints.

This legacy prompt is kept only for compatibility with older notes or references. It must not be used for new chapter state creation or formal next-chapter planning.

Hard rule:

```text
Approved formal chapter -> prompts/05_wiki_sync_after_chapter.md -> next chapter.
```
