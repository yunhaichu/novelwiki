# Chapter Brief Prompt

Use this prompt before drafting any chapter.

The chapter brief is a pre-draft gate. Do not write the chapter until the brief passes review.

## Inputs

Read:

- project direction
- current arc outline or chapter roadmap
- current novel style file
- current novel glossary / approved term list
- relevant character files
- previous chapter state
- relevant open questions
- `governance/longform_ai_inertia_check.md`
- `governance/low_anchor_phrase_check.md`

## Required Brief

Fill in:

```text
Chapter number:
Arc beat:
Main engine:
External event or pressure:
Protagonist goal:
Protagonist action:
Main obstacle:
Visible gain:
Cost or risk:
Cultivation relevance:
Who notices or acts:
Next hook:
```

## Anti-Repetition Plan

Fill in:

```text
Previous chapter skeleton:
Skeleton this chapter must not repeat:
How this chapter breaks that skeleton:
Time-order risk:
How the chapter avoids morning-noon-evening log structure:
```

## Terminology Plan

Fill in:

```text
Glossary terms likely to appear:
Possible low-anchor phrases to avoid:
Plain-language replacements:
```

## Character Concealment Plan

For each recurring side character in the chapter, fill in:

```text
Character:
What they want:
What they say on the surface:
What they do instead of saying the real emotion directly:
What they will not say:
```

## Physical State Plan

Fill in:

```text
Wound status:
Clothing/smell status:
Fatigue status:
Food/sleep status:
Details not to repeat:
New concrete detail if needed:
```

## Cultivation Conversion Plan

Fill in:

```text
Work/status event if any:
Cultivation-relevant conversion:
Three-month deadline relevance:
What changes in Zhou Ping's practice:
What remains unsolved:
```

## Output

Output only the brief.
