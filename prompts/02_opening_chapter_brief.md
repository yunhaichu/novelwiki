# Opening Chapter Brief Prompt

Use this prompt before drafting chapter 1-3, a new volume opening, a new major arc opening, or the first appearance of a major new system.

This brief is deliberately smaller than the normal chapter brief. Its purpose is reader entry, not full story-bible delivery.

Do not use this prompt for ordinary middle chapters unless the chapter functions as a new opening.

## Inputs

Read:

- approved type contract
- approved reader entry gate
- approved project outline if present
- approved volume plan if present
- approved chapter beat table if present
- current novel style file
- current glossary / approved term list
- relevant character files
- prior opening or golden-three design if present

Do not draft prose until this brief passes `governance/opening_chapter_brief_review.md`.

## Required Opening Brief

Fill in:

```text
Chapter number:
Opening function:
Primary reader promise:
One-sentence reader recap target:

Protagonist entry:
Immediate pressure:
Protagonist visible desire:
Obstacle:
Cost if no action:
One active choice:
Visible consequence:
One concrete memory point:
Who notices or reacts:
End hook:
```

## Reader Entry Control

Fill in:

```text
First 500 words must show:
1.
2.
3.

Information allowed now:
Information withheld:
Backstory limit:
Worldbuilding limit:
Terms forbidden in opening:
```

## Type Promise Control

Fill in:

```text
Declared primary genre promise:
How this opening makes it visible:
What type promise must not be falsely signaled:
Secondary genre if any:
How secondary genre supports, not replaces, the primary promise:
```

## Character Action Control

Fill in:

```text
Why must the protagonist act now?
What does the protagonist protect first in this scene?
Which flaw, fear, or pressure is triggered?
What wrong or partial choice can this produce?
What consequence does the choice create?
```

## Anti-Overload Rules

For opening chapters, do not:

- open with world explanation before pressure;
- introduce many names before the reader knows the protagonist;
- introduce many systems, rules, jobs, ranks, locations, or organizations before the reader understands the immediate trouble;
- rely on backstory labels instead of present pressure;
- use mystery as a substitute for stake;
- use setting mood as a substitute for story direction;
- let the protagonist only observe, process work, or serve as a camera.

## Output

Output only the opening chapter brief.
