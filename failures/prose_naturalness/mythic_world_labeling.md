# Failure Case: Mythic World Labeling

## Symptom

A mythic or fantasy scene uses names that are clear in planning but feel artificial in prose.

The name sounds like:

```text
map label
system UI
quest marker
setting card
outline category
```

## Examples

```text
White Bone Ridge West Road
patrol sequence
old-world rule
spirit corridor
fate gap
```

## Reject Rule

If a term sounds more like author-side classification than character-side speech or object marking, mark:

```text
Prose decision: NEEDS LINE REVISION
```

## Repair Strategy

Use rougher, shorter, more local wording.

Possible repairs:

- make the object label partial;
- let a character say the full location naturally;
- use a nickname;
- show direction through action;
- introduce formal names later through records, maps, officials, or inscriptions.

## Expected Decision

```text
Workflow decision: PASS if structure works
Prose decision: NEEDS LINE REVISION
```
