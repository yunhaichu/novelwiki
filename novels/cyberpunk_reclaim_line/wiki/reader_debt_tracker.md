# Reader Debt Tracker

Novel ID: `cyberpunk_reclaim_line`

## Instructions

This file tracks all outstanding reader debts across chapters.

- **Debt**: A question, tension, or payoff that was raised in a prior chapter but not yet answered.
- **Age**: Number of chapters since the debt was created.
- **Priority**: P0 (must resolve within 3 chapters), P1 (must resolve within 6 chapters), P2 (can remain open longer if tension is maintained).
- **Status**: active, partially_paid, paid, abandoned (with reason).

**Hard rule**: Before writing the next chapter's wiki sync, check this tracker. If any debt is P0 and age >= 3, or P1 and age >= 6, the next chapter must address it — either directly or through a meaningful shift in how it's framed.

If no debt meets the threshold, the next chapter must add at least one new debt to prevent flat accumulation.
