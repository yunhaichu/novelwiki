# Style

## Language

Use clear modern Chinese prose.

Prefer practical, visible, scene-grounded wording.

## Tone

Suspenseful but restrained.

The story should feel like ordinary workplace process becoming slightly wrong, not like grand prophecy or cosmic fate.

## Dialogue

Dialogue should come from complaint, delay, refusal, blame avoidance, work pressure, fear of responsibility, or narrow practical need.

Avoid characters explaining rules like a manual.

Avoid overly subtle half-said dialogue chains that make the conflict tiring to parse.

## Description

Use concrete objects and records:

- ticket ID
- timestamp
- callback note
- reply template
- quality score
- audit mark
- headset
- screen glow
- unread queue
- supervisor message

Do not use abstract labels as substitutes for story movement.

## Avoid

- 命运
- 旧秩序
- 暗流
- 真正的问题才刚刚开始
- 这不是结束，而是开始
- 心跳漏了一拍
- 说不清是什么感觉
- 平台背面
- 系统深处

These phrases are not absolutely banned, but they require explicit justification. Prefer visible action, object, choice, or consequence.
