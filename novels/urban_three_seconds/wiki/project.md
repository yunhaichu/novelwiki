# Project

Novel ID: `urban_three_seconds`

Working title: 《三秒钟》

Status: new test novel using Name & Term Gate and xiaobai prose flow.

## Type Promise

都市异能。

普通城市、普通人、普通工作。异能不宏大，不命名，不进体系，先从一件小事开始。

## Core Reader Promise

- 主角是普通夜班便利店员工，不是天才、富二代、神秘组织成员。
- 异能表现很小：他碰到别人刚用过的东西时，会看见几秒钟不属于现在的画面。
- 前期能力不能主动控制，不能看未来，不能读心，不能破案如神。
- 故事从老小区、便利店、派出所、学校周边这些普通场景展开。
- 每次使用能力都带来误会、身体不适或现实麻烦。

## Protagonist

Name: 周成

Age: 27

Starting role: 青禾便利店夜班员工，兼帮附近快递点临时看件。

Starting pressure: 工资低、夜班累、欠房租、怕丢工作。

Ability description:

他碰到别人刚用过的东西时，有时会看见一段很短的画面。画面不是现在发生的，通常只有几秒。

This ability has no approved name in the first three chapters.

## Core Contradiction

周成看见了别人看不见的几秒钟画面。

但他无法解释自己怎么知道。

如果他说出来，别人会觉得他偷看、跟踪、撒谎，甚至和事情有关。

如果他不说，有些人可能真的出事。

## Story Engine

```text
普通夜班生活
-> 捡到或碰到他人物品
-> 看见短画面
-> 误以为自己眼花
-> 小心试探
-> 帮人但说不清来源
-> 被怀疑 / 被牵连 / 获得小线索
```

## What This Story Is Not

- 不是系统流。
- 不是修炼体系。
- 不是异能组织开局。
- 不是警方破案外挂。
- 不是主角马上无敌。
- 不是玄乎设定名词堆砌。


## Genre Mode

Genre mode: 都市异能
Subgenre: 普通城市、普通人、普通工作，异能从一件小事开始
Primary pressure carrier: 误会、身体不适、现实麻烦，每次能力使用带来的代价
What scenes make readers continue: 老小区、便利店、派出所、学校周边的日常场景，主角用普通方法帮人
What scenes feel off-genre: 系统流、修炼体系、异能组织开局、警方破案外挂、主角马上无敌、玄乎设定名词堆砌
Key forbidden moves: 前期能力主动控制、看未来、读心、破案如神、能力不命名不进入体系

## First Arc Direction

周成在夜班时捡到一部手机，碰到手机后看见一个女孩在便利店后巷被人拽住包带。

他不知道这是不是幻觉。

第二天女孩家人找来，他才意识到那几秒钟可能是真的。

前三章目标：

- 让周成第一次发现能力；
- 让他试图用普通方法帮人；
- 让他因为说不清线索来源而被怀疑；
- 让能力保持弱、小、麻烦。
