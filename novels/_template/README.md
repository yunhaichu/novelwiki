# Novel Template

Each novel must be created from this template.

Every novel has an independent wiki, independent drafts, independent chapter states, and independent name registry.

Recommended structure:

```text
novels/<novel_id>/

wiki/
  project.md
  style.md
  name_registry.md
  timeline.md
  relationships.md
  foreshadowing.md

  characters/
    _template.md

  world/
    rules.md
    locations.md
    items.md

  chapter_states/
    _template.md

drafts/
  README.md
```

## Name Registry

Each novel must maintain its own `wiki/name_registry.md`.

The registry should keep character names simple, natural, and easy to remember.

Major character names should remain stable once approved.

## Independence Rule

This wiki is independent from all other novels.

Do not share character state, world rules, timelines, or draft assumptions across novels.
