# Body State Continuity

Use this file before drafting or reviewing each chapter.

Physical details should change over time. Do not repeat old wounds, smells, stains, fatigue, hunger, or pain as default atmosphere.

## After Chapter 004

### Zhou Ping

- Hand wound: mostly scabbed. It should not be a main pain detail unless reopened or used for a new consequence.
- Sleeve smell: washed down. It should no longer be repeated as a main sensory tag.
- Dirt/oil mud: not current unless a new scene creates it.
- Fatigue: normal servant-probation fatigue. Do not restate as default mood.
- Hunger: available as a chapter 5 issue only if connected to false heat during qi-guiding.
- Sleep: uncomfortable but not a primary detail unless the chapter uses sleep loss as plot pressure.

### Wang Man

- Shoulder strain from carrying water: can appear if it affects action, but do not repeat as generic suffering.
- Food bag: can appear only if it changes behavior, such as saving food, eating before practice, or hiding hunger.
- Anxiety: should be shown through action, timing, choices, or guarded speech, not direct emotional confession.

### Liu Er

- Carrying-water strain: may exist but should not be used only for easy punishment.
- Current function: watching, copying, joking, or spreading pressure around Zhou Ping's different treatment.

## Reuse Rule

A physical detail may return only if it does one of these:

- changes state
- affects a choice
- causes a social reaction
- affects qi-guiding practice
- creates or resolves a risk

Otherwise, omit it.

## Forbidden Default Reminders

Do not repeatedly use:

- hand wound as automatic grit
- sleeve smell as automatic low-status marker
- oil mud as automatic memory
- hunger as vague suffering
- fatigue as filler

## Chapter 005 Starting State

Chapter 5 should not rely on dirt or wound details.

The usable physical issue for chapter 5 is food/hunger only if it directly affects practice, especially false heat or body distraction during qi-guiding.
