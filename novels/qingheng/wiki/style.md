# Style

## Default Style

- clear main line
- grounded cultivation-world tone
- light humor in moderation
- restrained narration
- no modern knowledge
- no transmigration framing
- no pure technical explanation
- no overuse of aphoristic lines

## Humor Rules

Humor should come from character reactions, social mismatch, pressure, misunderstanding, and ordinary people facing a harsh system.

Humor should not erase danger or replace the main line.

Do not overuse one joke, object, or recurring reaction in a single chapter.

If a repeated object appears again, it must add new information, cost, choice, or relationship change.

## Long-form Risk Controls

Avoid:

- repeated task-of-the-day structure
- isolated mystery escalation without payoff
- protagonist solving every problem correctly
- too many polished closing lines
- overuse of short philosophical dialogue
- side characters becoming fixed joke machines
- dialogue becoming a repeated setup-and-response rhythm

## Chapter Memory Point

Each chapter should have one concrete memory point that belongs to this story.

The memory point should come from a specific system detail, object, choice, cost, relationship shift, or consequence.

It should not rely only on a polished line or a generic genre scene.

## Language Rules

Character speech should match status, age, fear, knowledge, and situation.

Avoid overly literary names and overly poetic dialogue.

Prefer action and consequence over summary lines.

Strong closing lines should be rare. A chapter should not depend on repeated aphoristic endings to feel polished.
