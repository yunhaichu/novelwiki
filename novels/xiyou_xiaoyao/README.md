# 西游小妖

Project ID: `xiyou_xiaoyao`

This is a test novel instance for the reader-entry workflow upgrade.

Genre:

```text
xianxia / mythic transmigration / survival counterattack / light comedy
```

Premise:

A modern person wakes in the world of Journey to the West as a nameless mountain-patrol little demon. He soon realizes the cave he belongs to is on the route that Sun Wukong will clear before Tang Sanzang arrives.

Workflow test goal:

Use this project to test whether the reader-entry workflow can handle a strong existing mythic setting without overloading chapter 1 with lore.

The opening must make clear:

- the protagonist is now a low-ranking little demon;
- he understands enough about Journey to the West to know the danger;
- his immediate problem is survival, not grand cultivation;
- the first choice has cost and consequence;
- comedy comes from role mismatch and practical survival, not random jokes.
