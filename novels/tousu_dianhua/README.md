# 投诉电话

Project ID: `tousu_dianhua`

This is a test novel instance for the reader-entry workflow upgrade.

Genre:

```text
urban rebirth / workplace counterattack / light realistic suspense
```

Workflow test goal:

Use this project to verify that the upgraded system can produce and review a chapter 1 opening where a first-time reader can understand:

- who the protagonist is;
- what immediate trouble exists;
- what the protagonist wants now;
- what can be lost soon;
- what the primary genre promise is;
- why chapter 2 is worth reading.
