# Opening Chapter Brief Review Gate

Use this before drafting chapter 1-3, a new volume opening, a new major arc opening, or the first appearance of a major new system.

The opening chapter brief must pass before prose drafting begins.

## Decision

Return one of:

```text
PASS
REVISE
```

## Required Checks

### 1. Reader Entry

The brief must make clear what a first-time reader should understand within the first 500 words.

Mark `REVISE` if it requires prior wiki knowledge.

### 2. Protagonist Entry

The protagonist must enter as a person under pressure.

Mark `REVISE` if the protagonist appears mainly as a job function, observer, or process executor.

### 3. Immediate Pressure

The opening must have a concrete pressure, imbalance, threat, desire gap, unfairness, or abnormality.

Mark `REVISE` if it starts with ordinary routine or setting explanation before pressure.

### 4. Visible Desire

The protagonist must visibly want something now.

Mark `REVISE` if the protagonist only reacts or hides information without a clear immediate want.

### 5. Cost of Inaction

The brief must state what can be lost soon.

Mark `REVISE` if the opening is only curious or atmospheric.

### 6. Active Choice

The opening must contain one active protagonist choice.

Mark `REVISE` if events only happen to the protagonist.

### 7. Visible Consequence

The active choice must create a visible consequence.

Mark `REVISE` if the chapter only reveals information without changing pressure, relation, risk, or next action.

### 8. Type Promise

The declared primary genre promise must be visible in the opening.

Mark `REVISE` if the brief pays only a secondary promise or uses setting as a substitute for type promise.

### 9. Information Load

The brief must delay unnecessary names, terms, rules, organizations, backstory, and secondary mysteries.

Mark `REVISE` if the reader has to process too many elements before knowing what to care about.

### 10. Concrete Memory Point

The opening must have one concrete memory point: object, action, danger, humiliation, choice, cost, relationship reaction, or abnormal evidence.

Mark `REVISE` if the memory point is only mood, theme, or polished narration.
