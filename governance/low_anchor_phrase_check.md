# Low-Anchor Phrase Check

Use this during chapter brief review and chapter review.

This check targets phrases that are not formal terms but still feel machine-made, pseudo-archaic, over-compressed, or weakly grounded.

## Core Rule

A phrase can be ordinary language and still fail.

Reject a phrase if it seems to exist mainly because it sounds genre-appropriate, compact, or polished, while real speakers or clear narration would use plainer wording.

## Red Flags

Mark REVISE when a draft relies on:

- pseudo-archaic quantities
- vague group labels
- compressed system-like labels
- polished contrast sentences
- repeated uncertainty tags
- abstract summary phrasing instead of action

## Examples To Avoid

Avoid these unless the surrounding scene makes them genuinely natural:

```text
多坐了几口气
几口气
几息
旧人
路子
在册上了
册上有人
不是木牌上，是册子上
像是有了个说法
这次，他是真的不知道
这一次，没人笑了
不是奖励
不算好事
门缝
```

## Plain Replacements

Use plainer wording:

```text
昨天是不是比前几天多坐了一会儿
已经在这里待过一个月的人
木牌背后已经刻过一道的人
赵管事在册子上写了你的名字
赵管事在你名字后面做了记号
这只木瓢能说明他是被派来的
他不知道
没人再接话
这事有用，也容易出错
机会很小
```

## Dialogue Test

For every suspicious line, ask:

```text
Would this character say this under pressure?
Would a reader pause to decode it?
Is the phrase compressing a world rule that should be shown plainly?
Can it be replaced by ordinary speech without losing meaning?
```

If yes, revise into ordinary language.

## Narration Test

For every suspicious narration phrase, ask:

```text
Is this a summary label?
Is it repeating a previous polished line pattern?
Does it sound like a motto rather than observation?
Does it explain the chapter's theme too neatly?
```

If yes, replace with action, object, consequence, or plain description.

## Decision Rule

Mark REVISE if:

- more than one low-anchor phrase appears in a chapter
- a key payoff depends on low-anchor wording
- a character speaks in labels rather than practical speech
- narration uses pseudo-archaic compression where ordinary wording is clearer
