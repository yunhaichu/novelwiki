# Style Banlist

This file tracks phrases, structures, and habits that repeatedly create AI taste.

These items are not always forbidden, but they require explicit justification.

## Low-Anchor Expressions

Avoid:

```text
几口气
多坐了几口气
几息
旧人
路子
在册上了
册上有人
有说法
这次他是真的不知道
这一次没人笑了
```

Prefer plain wording.

## Summary-Like Narration

Avoid narration that explains the chapter's theme too neatly.

Examples:

```text
这不是奖励
这是代价
真正的问题是
他终于明白
```

Prefer action, consequence, or observation.

## Repeated Skeletons

Watch for:

```text
receive task
mockery
cost
lesson payoff
```

and

```text
morning
midday
evening
after-event conversation
```

## Fixed Detail Loops

Do not reuse:

- hand wound
- sleeve smell
- oil mud
- hunger
- fatigue

as default atmosphere.

## Character Function Loops

Avoid:

- Wang Man = direct jealousy speaker
- Liu Er = mockery machine
- Liu Shun = explanation machine
- Zhao Steward = hidden evaluator who always appears at chapter end

Characters must act, not only symbolize a function.
