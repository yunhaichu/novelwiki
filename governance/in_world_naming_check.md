# In-World Naming Check

Use this during outline review, brief review, drafting, chapter review, and line revision.

This check prevents artificial names that sound like map labels, system UI, game quest markers, or author-side setting tags.

## Core Rule

A name should feel like something people inside the world would actually use.

It should pass at least one of these tests:

- ordinary speakers in the scene would say it;
- the object would realistically be labeled this way;
- a superior or institution would use this form;
- a low-status character would use a shorter local version;
- the name can be shortened naturally in dialogue;
- the name does not explain the world more clearly than the characters would.

## Suspicious Patterns

Usually review names that feel like:

```text
map label
system category
quest marker
rule label
symbolic theme label
polished worldbuilding tag
compressed lore phrase
```

Examples of risky forms:

```text
White Bone Ridge West Road
old-world rule
fate gap
order backside
forgotten truth
patrol sequence
spirit corridor
```

These may be valid in some contexts, but the scene must justify them.

## Repair Methods

Use one or more:

```text
shorten the label
make the object partially labeled
let a character say the place in rough local speech
replace formal map language with practical direction
turn the term into a visible object rather than a concept
introduce the formal name later through records, maps, or officials
```

Example repair pattern:

```text
Weak: The token reads White Bone Ridge West Road.
Better: The token reads West Road; another demon says it means the road west of White Bone Ridge.
```

## Review Rules

Mark `REVISE` if:

- a key term sounds like author classification rather than in-world wording;
- a low-status speaker uses a polished formal term without reason;
- a rough object carries a precise system label without explanation;
- the term exists mainly to make the reader understand the setting efficiently;
- the same naming pattern appears repeatedly across a chapter.
