# Object Function Review

Use this checklist during reality-causal preflight, scene convergence, drafting, and final review.

The purpose is to prevent random objects from becoming fake evidence, fake talismans, fake clues, fake hooks, or plot conveniences.

## Core Principle

```text
A key object must earn its importance.
```

An object may become important only if it naturally changes action, access, status, danger, relationship, resource, or knowledge in the current genre reality.

## Required Inputs

Read:

- genre mode contract;
- reality-causal preflight;
- scene convergence plan;
- chapter design;
- relevant item / object wiki if present;
- chapter draft if present.

## Hard Checks

### 1. Natural Importance Check

Why does this object matter before the author wants it to matter?

Mark `REVISE` if the object is important only because the plot says so.

### 2. Actor Interest Check

Who would naturally care about this object?

Who would ignore it?

Who would move, protect, take, hide, inspect, destroy, trade, or preserve it?

Mark `REVISE` if no actor has a natural reason to act on the object.

### 3. Function Check

What can this object change?

Valid functions include:

- proves position;
- proves contact;
- proves timing;
- changes access;
- gives resource;
- carries spiritual residue / craft trace / physical trace with genre support;
- creates debt;
- changes public standing;
- exposes contradiction;
- blocks action;
- enables next scene;
- becomes a contested possession.

Mark `REVISE` if the object only creates atmosphere.

### 4. Genre Support Check

Does the object's function fit the genre's reality?

For xianxia, ask:

- why this object can carry qi, residue, intent, talisman mark, blood, ownership, or lineage;
- who can detect that;
- why ordinary people did not notice;
- what cost or danger comes from keeping it.

### 5. Replaceability Check

If this object were removed, would the chapter still work?

If yes, either demote it to atmosphere or give it a concrete function.

### 6. Continuity Check

If the object is kept as important, can it appear again with consistent function?

Mark `REVISE` if the object is a one-scene trick.

## Repair Method

If the object fails, do one of:

- remove it;
- demote it to ordinary detail;
- replace it with a more natural object;
- establish a clear function before the scene depends on it;
- attach a cost, danger, or actor interest that makes its importance natural.
